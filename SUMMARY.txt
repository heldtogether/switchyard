📍 CONFIGURATION CLARITY - WHERE TO CHANGE SETTINGS:
================================================================================

ALL configuration is in ONE FILE: config.yaml (copy from config.example.yaml)

Key settings you WILL need to customize:

1. NFS Mount Path:
   executor.swarm.nfs_base_path: "/mnt/jobrunner"
   ↳ Change if your NFS is mounted elsewhere

2. S3 Storage:
   storage.endpoint: "https://s3.amazonaws.com"
   storage.access_key: "YOUR_KEY"
   storage.secret_key: "YOUR_SECRET"
   storage.bucket: "jobrunner"

3. Database:
   database.url: "postgres://user:pass@host:5432/dbname"

4. Redis:
   queue.url: "redis://host:6379/0"

5. API Key:
   api.auth.api_key: "your-secret-key"

6. Resource Defaults:
   executor.swarm.defaults.resources.cpu: "1.0"
   executor.swarm.defaults.resources.memory: "2g"
   executor.swarm.defaults.timeout: "1h"

Environment variables override config.yaml:
- EXECUTOR_NFS_BASE=/custom/path
- S3_ENDPOINT=https://minio.company.com
- API_KEY=production-key

Docker secrets (recommended for production):
- API_KEY_FILE=/run/secrets/api_key
- S3_ACCESS_KEY_FILE=/run/secrets/s3_access
- DATABASE_PASSWORD_FILE=/run/secrets/db_password
